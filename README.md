# X-Tweet Monorepo

This is the **X-Tweet Suite** monorepo, containing all services, packages, and modules for the X-Tweet project.

## Packages

- `x-tweet-admin` – Admin dashboard and management tools
- `x-tweet-ai` – AI features for moderation and recommendations
- `x-tweet-analytics` – Analytics and reporting
- `x-tweet-comments` – Commenting system
- `x-tweet-hashtags` – Hashtag management
- `x-tweet-media` – Media storage and processing
- `x-tweet-moderation` – Content moderation service
- `x-tweet-notifications` – Notifications system
- `x-tweet-notifications-queue` – Queue for notifications
- `x-tweet-payments` – Payment and subscriptions
- `x-tweet-profiles` – User profiles and settings
- `x-tweet-reactions` – Likes, reactions, and engagements
- `x-tweet-search` – Search engine
- `x-tweet-seo` – SEO enhancements
- `x-tweet-suggested` – Suggested content
- `x-tweet-supabase` – Supabase utilities
- `x-tweet-trending` – Trending topics
- `x-tweet-users` – User management
- `x-tweet-videos` – Video features
- `x-tweet-wallet` – Wallet system
- `x-tweet-webhooks` – Webhook integrations
